Nexus - Business Management System
Nexus is a Business Management System built with React and Vite.
It connects entrepreneurs with investors and helps manage business activities.
Features
User login system
Entrepreneur dashboard
Responsive design (mobile + desktop)
Clean UI using Tailwind CSS

Setup Instructions
Clone the repository:
   git clone https://github.com/Kiran-saba/Nexus.git
   Go to project folder:
   cd nexus
   Install dependencies:
   npm install
   Run the project:
   npm run dev
